📡 Endpoints

POST /v1/chat/completions OpenAI-compatible chat completions

Send chat messages and receive AI completions. Supports streaming.

curl Example

curl -X POST http://localhost:1430/v1/chat/completions \ -H "Content-Type: application/json" \ -H "Authorization: Bearer sk-xxxxx" \ -d '{ "model": "claude-sonnet-4", "messages": [ {"role": "user", "content": "Hello!"} ], "stream": true }'

Request Schema

{ "model": "string", // Model ID from /v1/models "messages": [ // Array of message objects { "role": "user", // "system" | "user" | "assistant" "content": "string" } ], "stream": true, // Enable SSE streaming "temperature": 0.7, // Optional: 0.0 - 2.0 "max_tokens": 4096 // Optional: max output tokens }

Response Schema

{ "id": "chatcmpl-xxx", "object": "chat.completion", "model": "claude-sonnet-4", "choices": [{ "index": 0, "message": { "role": "assistant", "content": "Hello! How can I help?" }, "finish_reason": "stop" }], "usage": { "prompt_tokens": 12, "completion_tokens": 8, "total_tokens": 20 } }
GET /v1/models List available models

Returns all available models with their routing tier and provider information.

curl http://localhost:1430/v1/models \ -H "Authorization: Bearer sk-xxxxx"
POST /v1/messages Anthropic Messages API format

Native Anthropic Messages API compatibility. Use this if your client expects the Anthropic format.

curl -X POST http://localhost:1430/v1/messages \ -H "Content-Type: application/json" \ -H "x-api-key: sk-xxxxx" \ -H "anthropic-version: 2023-06-01" \ -d '{ "model": "claude-sonnet-4", "max_tokens": 1024, "messages": [ {"role": "user", "content": "Hello!"} ] }'
